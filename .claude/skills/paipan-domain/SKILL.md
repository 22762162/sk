# 排盘领域知识速查

## 核心概念与判界规则

- **四柱**：年柱、月柱、日柱、时柱，各为一组干支（十天干 × 十二地支，六十甲子循环）。
- **年柱判界**：以**立春**为岁首。立春时刻之前出生归上一年柱，之后（含恰好等于）归当年。精度要求到秒——立春时刻由权威历源数据注入，代码禁止硬编码。
- **月柱判界**：以十二**节**（立春、惊蛰、清明…）为界，不是中气，也不是农历月。
- **日柱判界**：默认 23:00（晚子时起点）换日还是 00:00 换日是流派分歧 →「早晚子时」配置项，两个取值都必须可算。
- **时柱**：两小时一个时辰，以真太阳时为准（流派配置可选平太阳时）。
- **真太阳时** = 标准时 − 时区偏移修正 + 经度修正 + 均时差。修正可能导致跨日，从而改变日柱。
- **大运起运**：自出生时刻顺/逆数至最近的节，按「三天折一岁」折算（精确折算规则见 docs/paipan-spec.md，随规格演进）。顺逆由年干阴阳与性别决定。

## 工程红线（与 CLAUDE.md 一致）

- 历法事实（节气时刻、tzdata、夏令时）只能来自注入数据，禁止凭记忆写常量。
- 所有判界比较必须明确开闭区间，并在 spec 中有对应条款编号。
- 流派分歧做成配置项（枚举 + 默认值 + 出处），不做硬编码取舍。
